File Manager

Stagenow - 2.2

Overview

The File Manager allows an application to manage files on the device, for example to download a file from an FTP server and it save on the device.

Main Functionality

  • Copy a file from a location on a server (identified by a URI) to a location on the device (identified by a path and file name)
  • Copy a file from data embedded in the XML object to a location on the device (identified by a path and file name)
  • Transfer a file from one location on a device (identified by a path and file name) to another location on the same device (identified by a path and file name)
  • Delete a file from the device
  • Copy a folder or archive file containing multiple files to the device

File Action

Specify action to perform on file.

Parm Name: FileAction

Pivotal Parm: Yes

Versions: MX 4.2 +

Option Name Description Versions
1 Transfer\Copy File A file to be transfered or copied to the device. MX 4.2+
4 Delete File The batch file is encoded as binary. MX 4.2+
9 Download and Expand Archive File Download and expand an archive file on to the device. MX 5.0+

Target Access Method

Specify whether the batch file will be available on the device file system or embedded in the XML.

Shown if: File Action is transfer\copy

Parm Name: TargetAccessMethod

Pivotal Parm: No

Versions: MX 4.2 +

Option Name Description Versions
2 File in Device File System The destination will be a file in the device file system. MX 4.2+

Target Path and File name

Enter a path and file name on the device where the file will be copied or transferred to. Maximum of 255 characters.

Shown if: File Action is transfer\copy

Parm Name: TargetPathAndFileName

Pivotal Parm: No

Versions: MX 4.2 +

Target URI

Provide the URI to the file in the remote server.

Shown if: File Action is transfer\copy

Parm Name: TargetURI

Pivotal Parm: No

Versions: MX 4.2 +

Source URI

Enter a URI of the location of the file on the server.

Shown if: File Action is transfer\copy and source is remote server

Parm Name: SourceURI

Pivotal Parm: No

Versions: MX 4.2 +

Source Access

Specify where the source of te transfer\copy will be located.

Parm Name: SourceAccessMethod

Pivotal Parm: Yes

Versions: MX 4.2 +

Option Name Description Versions
1 File on remote server The source file will be located on a remote server. MX 4.2+
2 File on device file system The source file will be located on the device. MX 4.2+
3 File embedded in XML The source file will be embedded in the xml. MX 4.2+

Source Path and File

Enter the path and file name of the source file location on the device.

Shown if: File Action is transfer\copy and source is on device

Parm Name: SourcePathAndFileName

Pivotal Parm: No

Versions: MX 4.2 +

Embedded File Type

Choose what type of file is being embedded

Shown if: File Action is transfer\copy and source is embedded

Parm Name: SourceType

Pivotal Parm: No

Versions: MX 4.2 +

Option Name Description Versions
1 General Data File The embedded source file will be of any extension type. MX 4.2+
2 Android Application The embedded source file will be an Android application .APK. MX 4.2+
3 Android OS Update The embedded source file will be an Android OS Update in the form of a ZIP file. MX 4.2+
