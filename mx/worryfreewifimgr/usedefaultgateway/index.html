This Managed Configuration allows an Administrator using an EMM to configure the Type of Remote Gateway for an Analysis Session started for the Worry Free WiFi feature and should be specified only if the value Start is specified for the Managed Configuration Worry Free WiFi Configuration - Action.

If the value Default is chosen, then the Default Remote Gateway will be used.

If the value Custom is chosen, then a custom Remote Gateway will be used and the additional Managed Configuration Worry Free WiFi Configuration - Action Start Remote Gateway Type Custom Server Address MUST also be specified to supply the desired custom Gateway Server Address.
