Used to specify the Action to perform on an application or app-related setting. Actions can apply only to System apps, only to User apps, or to both (as noted in the table below).

Notes:

  • "Protected" Apps remain on the "Recent Apps" list after the list has been cleared using the "ClearRecentApps" Action.
    See Protected List Action parameter for more information.
  • When persisting the DisableGMSApps Action across an Enterprise Reset, the Action MUST be preceded by a bypass of the Android Setup Wizard to prevent unpredictable behavior.
    See Power Manager for more information.
  • Enable/Disable GMS Apps Action supported on SDM660-platform devices only.
