The RFID Manager (RfidMgr) is used to update, manage and control Zebra RFID peripherals.

Main Functionality

  • Update device firmware
  • Set the country of operation
  • Reset the radio
  • Reset to factory-default RFID settings
  • Enable/Disable:
    • Channel hopping
    • Configuration of advanced options
    • RFID manager UI
    • Settings control UI
    • Notifications for:
      • Attachment
      • Battery status
      • Device temperature
      • Successful decode
