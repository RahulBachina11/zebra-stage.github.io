Beacon Manager (BeaconMgr) controls the signal transmitted by the Bluetooth Low Energy (BLE) radio of a powered-off device to help locate the device using Zebra Device Tracker software running on a second device. Beacon Manager allows administrative control of the interval and power level of the transmitted signal to help balance power consumption and detection range. Beacons transmit non-connectible advertising packets.

Main Functionality

  • Turn On/Off:
    • Beacon transmission when device is powered off
    • Beacon transmission when device is in "Airplane mode"
  • Set the beacon transmission power level
  • Set the transmission interval
  • Set beacon parameters for:
    • Beacon type
    • AltBeacon universally unique identifier (UUID)
    • iBeacon UUID
    • Major ID
    • Minor ID
    • Custom beacon data
