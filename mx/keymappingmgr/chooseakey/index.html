Used to identify the Key for which an Override-Key Mapping will be added or modified. To add or modify an Override Mapping for a Key, specify a value that contains a Key Identifier listed in the table below.

Not all devices contain Keys that correspond to every Key Identifier in the table. Attempts to add an Override Mapping to a Key Mapping Table for a Key that is not present on a device, an error will be returned in the Result XML. To determine the supported Keys on a specific device, see the device documentation or query the KeyMappingMgr to acquire a list of Key Identifiers (see Queries, below).
