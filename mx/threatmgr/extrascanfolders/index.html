This is the On/Off switch for folder monitoring. This Extra Scan Folders feature monitors one or more folders on the device (including those in Android-protected areas) as specified in the Extra Scan Folders List parameter. When a change to a monitoried folder occurs, ThreatMgr broadcasts a "folderchange" Intent with the folder name and one of the Android FileObserver constants to describe the event. Note: This parameter does not detect Threats.
