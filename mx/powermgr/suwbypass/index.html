Controls whether to bypass the "Android device setup wizard" (also known as the "Welcome Screen") following an Enterprise Reset initiated through PowerMgr. Applies only on devices with Google Mobile Services (GMS) and only when Enterprise Reset is initiated by PowerMgr.

Important: This setting does not persist on the device. If SUW bypass is desired following an Enterprise Reset, this parameter must be included in the same Profile that is used to initiate the Enterprise Reset.
