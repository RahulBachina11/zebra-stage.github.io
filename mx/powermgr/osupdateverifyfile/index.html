Used to specify the path and filename of the OS update manifest (XML file) or .zip file in the device file system to be used to verify an OS update. This parameter checks required OS- and hardware-level device compatibility before allowing an OS update procedure to continue. Supported only on devices running Android 8.x Oreo and later.

See Overview section for more information about the manifest file.
