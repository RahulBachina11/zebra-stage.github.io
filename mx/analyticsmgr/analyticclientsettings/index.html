This is the On/Off switch for Analytics Client Settings, which allow AnalyticsMgr to control the ability of client components (such as Stats) to submit data and events to AnalyticsMgr for processing and transport. By enabling this parameter, administrators can set limits on incoming data and events using the throttling parameters (below), and can choose whether to allow the upload of those elements to the cloud. Analytics Client Settings can be used to prevent the Analytics Engine from being overwhelmed by client activities.
