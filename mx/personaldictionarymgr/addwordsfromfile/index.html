Used to specify the path and file name of a file on the device containing terms to be added to the Personal Dictionary. A note about performance: Duplicate checking will increase

Parm value input rules:

  • Accepts a string from 1-64 characters
  • Must contain full path and file name (i.e. /storage/sdcard1/NewTerms.txt).
  • Must be a plain text file
  • Each term must be on a separate line
