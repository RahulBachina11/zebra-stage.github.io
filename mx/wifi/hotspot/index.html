This parameter applies the hotspot configuration on the device. When enabled, the device uses the settings configured with the hotspot parameters below to act as an access point for nearby devices to connect to the internet, sharing the device's Wi-Fi or cellular connection. Hotspot functionality is supported only on Zebra ET50 and ET56 devices running Android 5.0 Lollipop.

Note: This parameter does not control activation of the hotspot feature on the device.
