This is the On/Off switch for the ability to limit cellular data on the device, and the selector for applying an Android-default or custom limitation. When Enabled using the Android default limit, the limit will be the greater of 5GB or 1.2 times the value specified in the Cellular Data Warning Value parameter to a maximum of 1TB. If the custom limit is selected, the limit will be the value specified in the Cellular Data Limit Value parameter.

Once the limit is reached, the cellular data connection is terminated. If no value is specified with the Cellular Data Limit Value parameter, this parameter is ignored and the previously selected Cellular Data Limit State is retained.
